## About Me

Electronic Engineer with experience in Mechatronics, specialized in **Rust** for embedded systems and backend development. I build IoT solutions with **ESP32** (RISC-V), async backend services with **Axum/Tokio**, and full-stack applications with **TypeScript/Next.js**.

Currently focused on bridging the gap between hardware and software using Rust across the entire stack.
